-List 
    A list is a sequence (an ordered collection/container) of any number of values. The values are often referred to as elements. They can be anything: numbers, strings, booleans, even lists! They can also be a mixture of types.
    To create a list directly, like above:
    Write some square brackets: []
    If you don't want an empty list, write some expressions inside to be the elements.
    Put commas (,) between elements to separate them.

    can be added like strings
    .append(element) to add an element at last
    
    -pop: Removes and returns an element at a given index using the syntax
        some_list.pop(index)
        Without an argument, i.e. just some_list.pop(), it will remove and return the last element.

    -remove: Removes the first occurrence of the given value using the syntax
        some_list.remove(value)

    -sorted: Takes an iterable and returns a list of the elements in order from smallest to largest, using the syntax
        sorted(some_list)

    -in: A comparison operator that checks if a value is in a list, using the syntax
        value in some_list

    -sum: Add up an iterable of numbers using the syntax
        sum(some_list)
        example sum([5, 3, 4]) output 12

    -count: Returns the number of times the argument appears in the list using the syntax
        some_list.count(value)

- none
    None is a special 'null' value which can't do anything interesting. 
    It's a common placeholder that represents the lack of a real useful value. 
    Functions that don't want to return anything return None by default. 
    If you see an error message about None or NoneType, it often means you assigned the wrong thing to a variable
