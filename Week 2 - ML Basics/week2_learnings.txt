-K means  
  K in K means is a free parameter and we need to specify it's value at start of code (k is number of clusters)
  first task is to identify two random points(if k=2, 3 if k=3 and so on) which are centre of clusters (centroid)
  then we calculate distance of each data point from centroids, the centroid most near to it is the cluster of which that data point is a part
  then we repeatedly change position of centroids till the time equilibrium is reached i.e. none of the data point change position
  elbow method to determine value of k
    we compute sum of square errors(SSE) of individual clusters then the total SSE is calculated
    then we plot it at varying k values and find elbow
